# Cómo hice el formulario?

Para llevar a cabo mi formulario, primero leí las instrucciones que nos proporcionó el profesor por medio del Github.

El **primer reto** que nos pidieron fue que cambiáramos el color al bloque principal de el formulario y yo lo hice, buscando un color propio en la paleta de colores. Seguido de esto, nos pide que aumentemos el factor de borde redondos y yo lo hice en la parte de radio, aumentando los px.

Claramente para llevar todo esto a cabo,pusimos nuestro respectivo código de **html** y **css** en el Visual Studio Code.

Luego algo muy importante es conectar el en código **html**, el código de **css**.

El **segundo reto** que nos propone el profesor, es agregarle a el formulario una pieza o un cuadro nuevo para llenar la información y yo lo hice agregándole en medio de las piezas "asunto" y "mensaje" un **div** con la clase **campo**, debajo de este, abrí un **label** en donde denominé que esta pieza se llama **Edad**, dentro de un **input**, denominé lo siguiente: que el tipo de texto sea número: **type=**,  con un  **placeholder=** para que en el cuadro a la persona le aparezca: Selecciona tu edad.Por último, con una regla de número **min=** de 20 y **max=** de 60.

El **tercer reto** nos pedía modificar el botón **Épico** y yo lo hice simplemente yendo a el archivo de css y en la parte casi final del código donde dice: **.btn-primario**, en la parte de **box-shadow** les aumenté los px.
