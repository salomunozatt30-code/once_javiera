# once_javiera

Para guardar la información o trabajos que hagamos en la media técnica con el profesor debemos hacer lo siguiente:

**Conocer mis credenciales en github**

Primero debemos ingresar a nuestro github con nuestro respectivo usuario y contraseña.

**Conocer mi URL de github**

Después debemos saber cuál es nuestro URL para ingresar fácilmente al github.
Mi URL es: https://github.com/salo.

**Cómo crear un repositorio en github**

Lo primero que debemos hacer estando ya en nuestra cuenta es ingresar a **repositorios**. Luego de esto le damos a **New** y a este le podemos dar un nombre (lo cual no es obligatorio). Creamos el repositorio, este nos lleva a Visual Studio Code en donde llenamos la información requerida.
